# aisabella.ai

Website von AIsabella – Strategisches KI-Sparring für Unternehmer:innen, Inhaber:innen und Führungskräfte.

Live: [aisabella.ai](https://aisabella.ai)

## Tech-Stack

- [Astro](https://astro.build/) – Static Site Generator
- TypeScript (strict)
- Hosting: GitHub Pages

## Lokale Entwicklung

Voraussetzung: Node.js ≥ 22.12.0

```bash
npm install
npm run dev
```

Dev-Server läuft auf http://localhost:4321.

## Build

```bash
npm run build
```

Erzeugt das statische Output in `dist/`.
